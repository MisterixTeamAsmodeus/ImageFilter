## Введение

ImageFilter - это мощное консольное приложение для обработки изображений, которое позволяет применять различные фильтры и эффекты к вашим фотографиям. Это руководство поможет вам освоить все возможности приложения и научиться эффективно обрабатывать изображения.

### Основные возможности

- **22 фильтра** в 5 категориях
- **Цепочки фильтров** - применение нескольких фильтров последовательно
- **Пакетная обработка** - обработка множества изображений за раз
  - Параллельная обработка нескольких изображений одновременно
  - Возобновление прерванной обработки
  - Улучшенное отображение прогресса (время, ETA, скорость)
- **Пресеты** - сохранение и переиспользование настроек
- **Многопоточность** - автоматическая оптимизация производительности
- **Поддержка форматов** - JPEG, PNG, BMP

## Быстрый старт

### Установка и сборка

```bash
# Клонируйте репозиторий (если еще не сделали)
git clone <repository-url>
cd ImageFilter

# Создайте директорию для сборки
mkdir build && cd build

# Настройте проект
cmake .. -DCMAKE_BUILD_TYPE=Release

# Соберите проект
cmake --build .

# Исполняемый файл будет в build/bin/ImageFilter
```

### Первая команда

Попробуйте применить простой фильтр:

```bash
./bin/ImageFilter input.jpg grayscale output.jpg
```

Это преобразует ваше изображение в оттенки серого.

### Получение справки

```bash
# Список всех доступных фильтров
./bin/ImageFilter --list-filters

# Информация о конкретном фильтре
./bin/ImageFilter --filter-info blur
```

## Подробные примеры использования

### Цветовые фильтры

#### 1. Преобразование в оттенки серого

**Базовое использование:**
```bash
./ImageFilter photo.jpg grayscale bw_photo.jpg
```

**С сохранением альфа-канала (для PNG):**
```bash
./ImageFilter photo.png grayscale bw_photo.png --preserve-alpha
```

**Результат:** Цветное изображение преобразуется в черно-белое с сохранением яркости.

#### 2. Эффект сепии (винтажный)

**Базовое использование:**
```bash
./ImageFilter photo.jpg sepia vintage_photo.jpg
```

**Результат:** Изображение получает теплый коричневатый оттенок, создавая эффект старинной фотографии.

#### 3. Инверсия цветов

**Базовое использование:**
```bash
./ImageFilter photo.jpg invert inverted_photo.jpg
```

**Результат:** Все цвета инвертируются, создавая негативный эффект.

#### 4. Изменение яркости

**Увеличение яркости:**
```bash
./ImageFilter dark_photo.jpg brightness bright_photo.jpg --brightness-factor 1.5
```

**Уменьшение яркости:**
```bash
./ImageFilter bright_photo.jpg brightness dimmed_photo.jpg --brightness-factor 0.8
```

**Рекомендуемые значения:**
- `0.7-0.9` - затемнение
- `1.0` - без изменений
- `1.1-1.5` - осветление
- `>1.5` - сильное осветление (может привести к переэкспонированию)

#### 5. Изменение контрастности

**Увеличение контраста:**
```bash
./ImageFilter flat_photo.jpg contrast contrasty_photo.jpg --contrast-factor 1.8
```

**Уменьшение контраста:**
```bash
./ImageFilter harsh_photo.jpg contrast soft_photo.jpg --contrast-factor 0.9
```

**Рекомендуемые значения:**
- `0.7-0.9` - уменьшение контраста (мягкое изображение)
- `1.0` - без изменений
- `1.2-1.8` - увеличение контраста
- `>2.0` - очень высокий контраст (может быть слишком резко)

#### 6. Изменение насыщенности

**Увеличение насыщенности:**
```bash
./ImageFilter dull_photo.jpg saturation vibrant_photo.jpg --saturation-factor 1.8
```

**Создание черно-белого (полное обесцвечивание):**
```bash
./ImageFilter photo.jpg saturation bw_photo.jpg --saturation-factor 0.0
```

**Рекомендуемые значения:**
- `0.0` - полное обесцвечивание (grayscale)
- `0.5-0.8` - приглушенные цвета
- `1.0` - без изменений
- `1.3-2.0` - усиление насыщенности
- `>2.0` - очень насыщенные цвета (может выглядеть неестественно)

### Геометрические фильтры

#### 7. Горизонтальное отражение

```bash
./ImageFilter photo.jpg flip_h mirrored_photo.jpg
```

**Результат:** Изображение отражается по вертикальной оси (левая и правая стороны меняются местами).

#### 8. Вертикальное отражение

```bash
./ImageFilter photo.jpg flip_v flipped_photo.jpg
```

**Результат:** Изображение отражается по горизонтальной оси (верх и низ меняются местами).

#### 9. Поворот на 90 градусов

**По часовой стрелке (по умолчанию):**
```bash
./ImageFilter photo.jpg rotate90 rotated_photo.jpg
```

**Против часовой стрелки:**
```bash
./ImageFilter photo.jpg rotate90 rotated_photo.jpg --counter-clockwise
```

**Результат:** Изображение поворачивается на 90 градусов. Размеры изображения меняются (ширина становится высотой и наоборот).

### Фильтры краёв и деталей

#### 10. Повышение резкости

**Базовое использование:**
```bash
./ImageFilter blurry_photo.jpg sharpen sharp_photo.jpg
```

**С настройкой силы:**
```bash
./ImageFilter photo.jpg sharpen sharp_photo.jpg --sharpen-strength 1.5
```

**Рекомендуемые значения:**
- `0.5-0.8` - легкое повышение резкости
- `1.0` - стандартная резкость (по умолчанию)
- `1.2-1.5` - сильное повышение резкости
- `>2.0` - очень резкое (может создать артефакты)

**Результат:** Усиливает края и детали изображения, делая его более четким.

#### 11. Детекция краёв

**Базовое использование:**
```bash
./ImageFilter photo.jpg edges edges_photo.jpg
```

**С настройкой чувствительности:**
```bash
./ImageFilter photo.jpg edges edges_photo.jpg --edge-sensitivity 0.7
```

**С выбором оператора:**
```bash
./ImageFilter photo.jpg edges edges_photo.jpg --edge-operator sobel
./ImageFilter photo.jpg edges edges_photo.jpg --edge-operator prewitt
./ImageFilter photo.jpg edges edges_photo.jpg --edge-operator scharr
```

**Рекомендуемые значения чувствительности:**
- `0.3-0.5` - низкая чувствительность (только сильные края)
- `0.5` - стандартная (по умолчанию)
- `0.6-0.8` - высокая чувствительность (больше деталей)
- `>0.8` - очень высокая (может быть слишком много шума)

**Результат:** Выделяет контуры и границы объектов, создавая эффект контурного рисунка.

#### 12. Эффект рельефа (Emboss)

**Базовое использование:**
```bash
./ImageFilter photo.jpg emboss embossed_photo.jpg
```

**С настройкой силы:**
```bash
./ImageFilter photo.jpg emboss embossed_photo.jpg --emboss-strength 1.5
```

**Результат:** Создает эффект объемного рельефа, как будто изображение выгравировано.

#### 13. Выделение контуров

```bash
./ImageFilter photo.jpg outline outlined_photo.jpg
```

**Результат:** Выделяет контуры объектов, создавая эффект обводки.

### Фильтры размытия и шума

#### 14. Размытие по Гауссу

**Базовое использование:**
```bash
./ImageFilter photo.jpg blur blurred_photo.jpg
```

**С настройкой радиуса:**
```bash
# Легкое размытие
./ImageFilter photo.jpg blur soft_photo.jpg --blur-radius 2.0

# Среднее размытие
./ImageFilter photo.jpg blur blurred_photo.jpg --blur-radius 5.0

# Сильное размытие
./ImageFilter photo.jpg blur very_blurred_photo.jpg --blur-radius 10.0
```

**Рекомендуемые значения:**
- `1.0-3.0` - легкое размытие (размытие фона)
- `3.0-7.0` - среднее размытие
- `7.0-15.0` - сильное размытие
- `>15.0` - очень сильное размытие

**Результат:** Создает плавное, естественное размытие изображения. Идеально для размытия фона или создания эффекта глубины резкости.

#### 15. Размытие по прямоугольнику (Box Blur)

**Базовое использование:**
```bash
./ImageFilter photo.jpg box_blur box_blurred_photo.jpg
```

**С настройкой радиуса:**
```bash
./ImageFilter photo.jpg box_blur box_blurred_photo.jpg --box-blur-radius 7
```

**Результат:** Более быстрое, но менее качественное размытие по сравнению с Gaussian blur. Подходит для предварительной обработки.

#### 16. Размытие движения

**Базовое использование:**
```bash
./ImageFilter photo.jpg motion_blur motion_photo.jpg
```

**С настройкой длины и угла:**
```bash
# Горизонтальное размытие движения
./ImageFilter car.jpg motion_blur motion_horizontal.jpg --motion-blur-length 20 --motion-blur-angle 0

# Диагональное размытие движения
./ImageFilter car.jpg motion_blur motion_diagonal.jpg --motion-blur-length 15 --motion-blur-angle 45

# Вертикальное размытие движения
./ImageFilter runner.jpg motion_blur motion_vertical.jpg --motion-blur-length 25 --motion-blur-angle 90
```

**Рекомендуемые значения:**
- **Длина:** 5-30 пикселей (больше = сильнее эффект движения)
- **Угол:** 0-360 градусов (0 = горизонтально, 90 = вертикально)

**Результат:** Имитирует эффект движения камеры или объекта, создавая размытие в определенном направлении.

#### 17. Медианный фильтр

**Базовое использование:**
```bash
./ImageFilter noisy_photo.jpg median cleaned_photo.jpg
```

**С настройкой радиуса:**
```bash
# Легкое удаление шума
./ImageFilter noisy_photo.jpg median cleaned_photo.jpg --median-radius 1

# Среднее удаление шума
./ImageFilter noisy_photo.jpg median cleaned_photo.jpg --median-radius 2

# Сильное удаление шума
./ImageFilter very_noisy_photo.jpg median cleaned_photo.jpg --median-radius 3
```

**Рекомендуемые значения:**
- `1` - легкое удаление шума
- `2` - стандартное (по умолчанию)
- `3-5` - сильное удаление шума (может размыть детали)

**Результат:** Эффективно удаляет шум и артефакты, сохраняя края объектов. Идеально для очистки фотографий с высоким ISO.

#### 18. Добавление шума

**Базовое использование:**
```bash
./ImageFilter clean_photo.jpg noise noisy_photo.jpg
```

**С настройкой интенсивности:**
```bash
# Легкий шум
./ImageFilter photo.jpg noise noisy_photo.jpg --noise-intensity 0.05

# Средний шум
./ImageFilter photo.jpg noise noisy_photo.jpg --noise-intensity 0.1

# Сильный шум
./ImageFilter photo.jpg noise noisy_photo.jpg --noise-intensity 0.2
```

**Рекомендуемые значения:**
- `0.01-0.05` - легкий шум (эффект зерна пленки)
- `0.1` - стандартный (по умолчанию)
- `0.15-0.3` - сильный шум
- `>0.3` - очень сильный шум (может испортить изображение)

**Результат:** Добавляет случайный шум к изображению. Может использоваться для создания эффекта старинной фотографии или для тестирования алгоритмов удаления шума.

### Стилистические фильтры

#### 19. Постеризация

**Базовое использование:**
```bash
./ImageFilter photo.jpg posterize posterized_photo.jpg
```

**С настройкой количества уровней:**
```bash
# Сильная постеризация (мало цветов)
./ImageFilter photo.jpg posterize strong_posterize.jpg --posterize-levels 4

# Средняя постеризация
./ImageFilter photo.jpg posterize medium_posterize.jpg --posterize-levels 8

# Легкая постеризация (много цветов)
./ImageFilter photo.jpg posterize light_posterize.jpg --posterize-levels 16
```

**Рекомендуемые значения:**
- `2-4` - очень сильная постеризация (плакатный эффект)
- `4-8` - средняя постеризация (художественный эффект)
- `8-16` - легкая постеризация
- `16-64` - минимальная постеризация

**Результат:** Уменьшает количество цветовых уровней, создавая эффект плаката или художественной печати.

#### 20. Пороговая бинаризация

**Базовое использование:**
```bash
./ImageFilter photo.jpg threshold binary_photo.jpg
```

**С настройкой порога:**
```bash
# Для светлых изображений
./ImageFilter light_photo.jpg threshold binary_photo.jpg --threshold-value 180

# Стандартный порог
./ImageFilter photo.jpg threshold binary_photo.jpg --threshold-value 128

# Для темных изображений
./ImageFilter dark_photo.jpg threshold binary_photo.jpg --threshold-value 80
```

**Рекомендуемые значения:**
- `50-100` - для темных изображений
- `100-150` - для средних изображений
- `128` - стандартный (по умолчанию)
- `150-200` - для светлых изображений

**Результат:** Преобразует изображение в черно-белое по пороговому значению. Все пиксели выше порога становятся белыми, ниже - черными.

#### 21. Виньетирование

**Базовое использование:**
```bash
./ImageFilter photo.jpg vignette vignetted_photo.jpg
```

**С настройкой силы:**
```bash
# Легкое виньетирование
./ImageFilter photo.jpg vignette vignetted_photo.jpg --vignette-strength 0.3

# Среднее виньетирование
./ImageFilter photo.jpg vignette vignetted_photo.jpg --vignette-strength 0.5

# Сильное виньетирование
./ImageFilter photo.jpg vignette vignetted_photo.jpg --vignette-strength 0.8
```

**Рекомендуемые значения:**
- `0.2-0.4` - легкое виньетирование (естественный эффект)
- `0.5` - стандартное (по умолчанию)
- `0.6-0.8` - сильное виньетирование (драматический эффект)
- `>0.8` - очень сильное (может быть слишком темно)

**Результат:** Затемняет края изображения, создавая эффект фокусировки внимания на центре. Популярный эффект в фотографии.

## Категории фильтров

### Цветовые фильтры

Используются для изменения цветовых характеристик изображения:
- `grayscale` - преобразование в оттенки серого
- `sepia` - эффект сепии
- `invert` - инверсия цветов
- `brightness` - изменение яркости
- `contrast` - изменение контрастности
- `saturation` - изменение насыщенности

**Когда использовать:** Для цветокоррекции, создания художественных эффектов, подготовки изображений для печати.

### Геометрические фильтры

Используются для изменения геометрии изображения:
- `flip_h` - горизонтальное отражение
- `flip_v` - вертикальное отражение
- `rotate90` - поворот на 90 градусов

**Когда использовать:** Для исправления ориентации, создания зеркальных эффектов, подготовки изображений для публикации.

### Фильтры краёв и деталей

Используются для работы с краями и деталями изображения:
- `sharpen` - повышение резкости
- `edges` - детекция краёв
- `emboss` - эффект рельефа
- `outline` - выделение контуров

**Когда использовать:** Для улучшения резкости, выделения деталей, создания художественных эффектов, анализа изображений.

### Фильтры размытия и шума

Используются для размытия и работы с шумом:
- `blur` - размытие по Гауссу
- `box_blur` - размытие по прямоугольнику
- `motion_blur` - размытие движения
- `median` - медианный фильтр (удаление шума)
- `noise` - добавление шума

**Когда использовать:** Для размытия фона, удаления шума, создания эффектов движения, художественной обработки.

### Стилистические фильтры

Используются для создания художественных эффектов:
- `posterize` - постеризация
- `threshold` - пороговая бинаризация
- `vignette` - виньетирование

**Когда использовать:** Для создания художественных эффектов, стилизации изображений, подготовки для печати.

## Практические сценарии

### Сценарий 1: Улучшение качества фотографии

**Задача:** Улучшить качество слегка размытой и тусклой фотографии.

**Решение:**
```bash
./ImageFilter photo.jpg "sharpen,contrast,brightness" enhanced_photo.jpg \
    --sharpen-strength 1.2 \
    --contrast-factor 1.3 \
    --brightness-factor 1.1
```

**Объяснение:**
1. `sharpen` - повышает резкость деталей
2. `contrast` - увеличивает контрастность для более выразительного изображения
3. `brightness` - слегка осветляет изображение

### Сценарий 2: Создание винтажного эффекта

**Задача:** Преобразовать современную фотографию в старинную.

**Решение:**
```bash
./ImageFilter modern_photo.jpg "sepia,noise,vignette" vintage_photo.jpg \
    --noise-intensity 0.12 \
    --vignette-strength 0.6
```

**Объяснение:**
1. `sepia` - создает теплый коричневатый оттенок
2. `noise` - добавляет зерно, имитирующее старую пленку
3. `vignette` - затемняет края для драматического эффекта

### Сценарий 3: Подготовка изображения для веб-публикации

**Задача:** Оптимизировать большое изображение для веб-сайта.

**Решение:**
```bash
./ImageFilter large_photo.jpg "sharpen,contrast" web_photo.jpg \
    --jpeg-quality 80
```

**Объяснение:**
1. `sharpen` - улучшает резкость (важно для веб)
2. `contrast` - делает изображение более выразительным
3. `--jpeg-quality 80` - хороший баланс между качеством и размером файла

### Сценарий 4: Создание черно-белого портрета

**Задача:** Создать выразительный черно-белый портрет с высокой контрастностью.

**Решение:**
```bash
./ImageFilter portrait.jpg "grayscale,contrast,sharpen" bw_portrait.jpg \
    --contrast-factor 1.8 \
    --sharpen-strength 1.3
```

**Объяснение:**
1. `grayscale` - преобразует в черно-белое
2. `contrast` - увеличивает контрастность для драматического эффекта
3. `sharpen` - подчеркивает детали лица

### Сценарий 5: Удаление шума с фотографии

**Задача:** Очистить фотографию с высоким ISO от шума.

**Решение:**
```bash
./ImageFilter noisy_photo.jpg "median,sharpen" cleaned_photo.jpg \
    --median-radius 2 \
    --sharpen-strength 1.1
```

**Объяснение:**
1. `median` - эффективно удаляет шум, сохраняя края
2. `sharpen` - восстанавливает резкость после удаления шума

### Сценарий 6: Создание художественного эффекта

**Задача:** Создать плакатный художественный эффект.

**Решение:**
```bash
./ImageFilter photo.jpg "posterize,saturation,contrast" artistic_photo.jpg \
    --posterize-levels 8 \
    --saturation-factor 1.6 \
    --contrast-factor 1.5
```

**Объяснение:**
1. `posterize` - уменьшает количество цветов
2. `saturation` - увеличивает насыщенность для ярких цветов
3. `contrast` - усиливает контрастность

### Сценарий 7: Пакетная обработка фотографий

**Задача:** Обработать все фотографии в директории одним набором фильтров.

**Решение:**
```bash
./ImageFilter --batch \
    --input-dir ./raw_photos \
    --output-dir ./processed_photos \
    --recursive \
    --pattern "*.jpg" \
    "sharpen,contrast,brightness" \
    --contrast-factor 1.2 \
    --brightness-factor 1.1
```

**Объяснение:**
- `--batch` - включает пакетный режим
- `--input-dir` - исходная директория
- `--output-dir` - директория для результатов
- `--recursive` - обрабатывает поддиректории
- `--pattern "*.jpg"` - обрабатывает только JPEG файлы

**Особенности:**
- **Параллельная обработка** - автоматически использует все доступные ядра процессора
- **Улучшенный прогресс** - отображает процент, время, ETA и скорость обработки:
  ```
  [ 45%] [23/50] photo_023.jpg | Время: 2м 15с | Осталось: 2м 45с | 0.17 файл/с
  ```

### Сценарий 7a: Возобновление прерванной пакетной обработки

**Задача:** Продолжить обработку большого количества файлов после прерывания.

**Решение:**
```bash
# Первый запуск - обработка с сохранением состояния
./ImageFilter --batch \
    --input-dir ./raw_photos \
    --output-dir ./processed_photos \
    --recursive \
    --pattern "*.jpg" \
    "sharpen,contrast" \
    --resume-state ./batch_state.txt

# Если обработка прервана (например, Ctrl+C), при следующем запуске
# с тем же файлом состояния уже обработанные файлы будут пропущены
./ImageFilter --batch \
    --input-dir ./raw_photos \
    --output-dir ./processed_photos \
    --recursive \
    --pattern "*.jpg" \
    "sharpen,contrast" \
    --resume-state ./batch_state.txt
```

**Объяснение:**
- `--resume-state` - файл для сохранения состояния обработки
- Состояние сохраняется автоматически каждые 10 обработанных файлов
- При возобновлении программа проверяет, какие файлы уже обработаны, и пропускает их
- Это особенно полезно при обработке тысяч файлов, когда процесс может быть прерван

### Сценарий 8: Использование пресетов

**Задача:** Применить одинаковую обработку к множеству фотографий.

**Шаг 1: Создание пресета**
```bash
./ImageFilter sample.jpg "grayscale,sharpen,vignette" output.jpg \
    --save-preset bw_enhanced \
    --sharpen-strength 1.3 \
    --vignette-strength 0.5
```

**Шаг 2: Применение пресета**
```bash
./ImageFilter photo1.jpg output1.jpg --preset bw_enhanced
./ImageFilter photo2.jpg output2.jpg --preset bw_enhanced
./ImageFilter photo3.jpg output3.jpg --preset bw_enhanced
```

**Объяснение:**
- Пресеты позволяют сохранить сложные настройки и применять их многократно
- Особенно полезно для пакетной обработки с одинаковыми настройками

## Советы и рекомендации

### Порядок применения фильтров

Правильный порядок фильтров в цепочке важен для получения лучших результатов:

1. **Геометрические преобразования** (flip, rotate) - применяйте первыми
2. **Удаление шума** (median) - если нужно, применяйте рано
3. **Цветовые коррекции** (brightness, contrast, saturation) - применяйте после геометрических
4. **Фильтры краёв** (sharpen, edges) - применяйте после цветовых коррекций
5. **Стилистические эффекты** (vignette, posterize) - применяйте в конце

**Пример правильного порядка:**
```bash
./ImageFilter photo.jpg "rotate90,brightness,contrast,sharpen,vignette" result.jpg
```

### Работа с параметрами

1. **Начинайте с малых значений** - постепенно увеличивайте параметры до получения желаемого эффекта
2. **Экспериментируйте** - разные изображения требуют разных значений
3. **Используйте `--filter-info`** - для проверки допустимых диапазонов параметров
4. **Сохраняйте пресеты** - для успешных комбинаций параметров

### Оптимизация размера файла

1. **Используйте JPEG для фотографий** - меньший размер файла
2. **Настройте качество JPEG:**
   - Веб: `--jpeg-quality 75-85`
   - Печать: `--jpeg-quality 90-100`
   - Миниатюры: `--jpeg-quality 60-75`
3. **Используйте PNG только когда нужно:**
   - Прозрачность (альфа-канал)
   - Максимальное качество без потерь

### Производительность

1. **Используйте Release сборку** для максимальной производительности
2. **Пакетная обработка** эффективнее, чем обработка по одному файлу
3. **Параллельная обработка** - в пакетном режиме автоматически используется многопоточность:
   - Программа использует все доступные ядра процессора
   - Несколько изображений обрабатываются одновременно
   - Значительно ускоряет обработку больших объемов файлов
4. **Возобновление обработки** - для больших задач используйте `--resume-state`:
   - Позволяет продолжить прерванную обработку
   - Автоматически пропускает уже обработанные файлы
   - Состояние сохраняется каждые 10 файлов
2. **Пакетная обработка** эффективнее, чем обработка по одному файлу
3. **Большие радиусы фильтров** (blur, median) требуют больше времени
4. **Длинные цепочки фильтров** обрабатываются дольше
